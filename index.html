HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The following bullet points describe child elements often found inside the head element
  • The title element defines the words in the title bar
  • The style element defines the CSS styles
  • The base element defines the base URL
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as the character set, description, keywords, author and viewport
  • The script element embeds client-side scripts in the HTML document

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • text-align centers the text on the page.
  • box-shadow adds shadow effects around an element's frame
  • The CSS box model shows the structure of every HTML element.
  • An example of a CSS rule is div {color:blue}
  • div is a selector which defines the element to which the rules apply.
  • color is a CSS property
  • blue is the value of that property
  • color:blue; is called a declaration

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch: gives a list of all branches
  • git merge [branch]: merges specified branch's history into the current one
  • git add [file]: adds a file to your next commit
  • git reset [file]: unstages (unmarks for commit) a file
  • git diff: shows what is changed but not staged
  • git diff --staged: shows what is staged but not yet committed
